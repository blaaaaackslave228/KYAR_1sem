Grids

Grid-верстка - это мощная система CSS, предназначенная для двумерного макета ваших страниц.
Grid предлагает функции автоматического размещения, которые упрощают процесс размещения элементов.
Grid позволяет создавать сложные адаптивные макеты с помощью медиа-запросов и процентного масштабирования.

Вы можете создавать вложенные grids для более сложных макетов.
